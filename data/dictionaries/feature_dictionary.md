# Feature Dictionary
| Feature | Type | Description |
| --- | --- | --- |
| RowNumber | string | Row number |
| CustomerId | integer | Unique 8-digit identification for each number |
| Surname | string | Surname of customer |
| CreditScore | integer | Customer's credit score (350-850) |
| Geography | string | Customer's location (France/Germany/Spain) |
| Gender | string | Customer's gender (Male/Female) |
| Age | integer | Customer's age (18-92) |
| Tenure | integer | Number of years the customer has been a client of the bank (0-10) |
| Balance | float | Amount of balance in customer's bank account (0-250898.09)
| NumOfProducts | integer | Number of products that a customer has purchased through the bank (1-4) |
| HasCrCard | integer | Denotes whether or not a customer has a credit card (0=No,1=Yes) |
| IsActiveMember | integer | Denotes whether the customer is active (0=No, 1=Yes) |
| EstimatedSalary | float | Estimated salary of customer (11.58-199992.48) |
| Exited | integer | Whether or not the customer has left the bank (0=No, 1=Yes) |
